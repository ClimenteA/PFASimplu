# Instalare pt. developeri

Aplicatia este facuta in Python, Django cu SQLite DB. 

- clone repo;
- `virtualenv .venv`;
- `source .venv/bin/activate`;
- `pip install -r requirements.txt`;
- `python manage.py makemigrations`;
- `python manage.py migrate`;
